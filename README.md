# Real-Time Cyber Attack Prediction Using Machine Learning

## Project Overview
This project uses Machine Learning and Wazuh SIEM logs to detect and predict cyber attacks in real time within on-premise environments.

## Features
- Real-time threat detection
- Log analysis using Wazuh SIEM
- Machine Learning-based attack prediction
- Anomaly detection
- Alert generation for security incidents

## Technologies Used
- Python
- Machine Learning
- Wazuh SIEM
- Random Forest
- JSON
- Jupyter Notebook

## Project Files
- ml_wazuh.py
- predict.py
- Wazuh_SIEM_RandomForest.py
- Wazuh_SIEM_RandomForest.ipynb
- alerts.json
